## Projects

### 01 · Global Developer Ecosystem Trends — Stack Overflow Survey Analysis
**Tools:** Python · Pandas · Scikit-learn · IBM Cognos · Matplotlib · Seaborn

A rigorous end-to-end data analysis of the Stack Overflow Annual Developer Survey exploring the current state of the global engineering workforce. The goal: identify market standards, map future technology trends, and determine the skill profile of a high-value developer.

**Analytical Highlights:**
- Removed 20 hidden duplicates using unique identifier logic — standard deduplication checks were insufficient
- Built a custom Split-and-Explode pipeline to tokenize multi-value technology columns (e.g. "Python;SQL;Java") for accurate frequency analysis
- Applied KNN imputation ($k=5$) to predict missing AI tool adoption based on work experience and role similarity
- Explicitly chose NOT to impute 40% missing salary data — used IQR filtering instead to avoid fabricating financial insights

**Key Findings:**
- JavaScript, Python, and SQL form the non-negotiable "Golden Triangle" of modern tech skills
- TypeScript and Go show the highest Interest Gap (future demand > current usage) — signaling a market shift toward type safety and high-performance microservices
- Salary growth is steepest in the first 0–10 years of experience, then plateaus — senior income growth is driven by role changes, not tenure
- Skills-based hiring is prevalent; Master's degrees show minimal salary advantage in pure dev roles

**Key Concepts:** KNN Imputation · Feature Engineering · Exploratory Data Analysis · IBM Cognos Dashboarding · Executive Presentation

### 02 · Olist E-Commerce Sales Performance Analysis
**Tools:** MySQL · Python · Tableau Public

A full SQL analysis of 99,441 real orders from Olist, Brazil's largest e-commerce marketplace. Goes beyond surface-level metrics — this project delivers actionable business intelligence across revenue, logistics, customer behaviour, and seller risk.

| Metric | Result |
|---|---|
| Total Revenue Analysed | R$15.4M |
| Orders Covered | 99,441 |
| Unique Customers | 93,358 |
| On-Time Delivery Rate | 91.9% |
| SQL Queries Written | 13 |

**Business Questions Answered:**
- How did revenue grow over 24 months, and what drove the peaks?
- Which states and categories are the top revenue contributors?
- How are customers segmented by lifetime spend?
- Which sellers pose a delivery risk and should be reviewed?
- Where should marketing focus to close the upsell gap?

**Key SQL Concepts:** Multi-table JOINs · CTEs · `LAG()` · `RANK()` · `NTILE()` · `CASE WHEN` segmentation · `DATEDIFF` · `COALESCE`

**Key Finding:** Revenue grew 137% over 24 months, peaking during Black Friday 2017. Only 3.1% of customers are repeat buyers — a significant retention opportunity for the business.
